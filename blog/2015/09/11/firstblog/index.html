Octopress搭建静态博客网站

导语

Octopress是一个基于 Jekyll 的静态博客站点生成系统，它很大程度上简化了用 Jekyll 搭建博客的过程。

一、Octopress的优势

命令行操作 纯文本写 博客定制性高 纯静态 版本化管理 迁移成本低 简洁的Ruby框架 Markdown语法

二、搭建Octopress博客需要了解的知识

  • Ruby(其实会几个命令即可，后续会讲到)
  • Markdown语法
  • Github(部署博客用，会简单使用即可）
  • Git(会一些简单的Git命令)

三、准备工作

系统：本机系统是window8 64位，window7以上均可。下载以下软件
Git    Ruby   DevKit   Python   Awesomium   Markdownpad

  • Git:版本管理工具，将代码托管到GitHub
  • Ruby+DevKit：生成静态网页
  • MarkdownPad：Window下Markdown语法编辑器
  • Awesomium：实现MarkdownPad实时预览功能
